Создать договор для клиента

Администратор заполняет данные договора и формирует документ для подписания. Часть параметров может быть предзаполнена по профилю клиента и выбранным тарифам.

После выбора клиента можно подтянуть его паспортные и контактные данные из профиля.

В зависимости от типа клиента требуется разный набор реквизитов: для ФЛ — паспортные данные, для ЮЛ — реквизиты организации по законодательству РФ.
