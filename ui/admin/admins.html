Персональные данные сотрудника (ФЗ‑152)

Отражаются основные категории персональных данных сотрудника, обрабатываемых оператором в рамках трудовых отношений и администрирования системы.

Категории ПД:

  • идентификационные данные (ФИО, должность);
  • контактные данные (служебный телефон, служебный e‑mail);
  • данные об учётной записи (логин, роли, история входов);

Правовое основание обработки:

  • исполнение трудового договора / договора ГПХ с сотрудником;
  • локальные нормативные акты оператора (положение о защите ПД, регламент доступа);

Статус согласий:

  • Согласие на обработку ПД сотрудника: получено при приёме на работу;
  • Согласие на получение служебных уведомлений: активно.
