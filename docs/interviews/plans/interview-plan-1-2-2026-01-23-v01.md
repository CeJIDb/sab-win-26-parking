# План интервью №1.2

> Каноничный план интервью, нормализованный и сохраненный в основной структуре репозитория.

## Цели

1. Валидация схемы текущих бизнес-процессов AS-IS, подготовленной командой по итогам интервью №1.1.
2. Детализация сценариев въезда на парковку:
   - для физических лиц;
   - для юридических лиц;
   - для новых и действующих клиентов.
3. Уточнение процессов навигации и размещения автомобилей на парковке.
4. Уточнение процессов оплаты, документооборота и выезда, если останется время.
5. Сбор обратной связи заказчика по потенциальным точкам автоматизации.

## Введение

1. Приветствие и проверка связи.
2. Подтверждение согласия заказчика на запись встречи.
3. Краткое напоминание контекста:
   - интервью проводится для валидации и уточнения схемы бизнес-процессов;
   - основа обсуждения - диаграммы, подготовленные командой по итогам предыдущей встречи.
4. Озвучивание целей текущей встречи и планируемых блоков обсуждения.

## Основная часть

### Валидация бизнес-процесса "Въезд на парковку"

1. Новый клиент:
   - развилка сценария на краткосрочную парковку и долгосрочный абонемент;
   - процесс для физических лиц: оформление договора при необходимости, оплата, выдача проксимити-карты, въезд на парковку;
   - процесс для юридических лиц: согласование условий, заключение договора, фиксация списка ТС, особенности предоплаты и постоплаты.
2. Действующий клиент:
   - проверка наличия действующего договора;
   - проверка актуальности контактных данных.
3. Въезд юридического лица:
   - проверка номера автомобиля через камеру или вручную охранником;
   - действия при отсутствии номера в базе: проверка через управляющего, согласование разового проезда, отказ во въезде без подтверждения.
4. Въезд физического лица:
   - использование проксимити-карты;
   - действия при проблемах с картой: проверка договора и оплаты, перепрограммирование карты, выдача новой карты;
   - процедура при утере или отсутствии карты: ручная идентификация клиента, условия выдачи новой карты.

### Процессы оплаты и документооборота

1. Фиксация успешной оплаты и снятие задолженности.
2. Взаимодействие с бухгалтерией:
   - получение информации об оплатах юридических лиц;
   - текущий ручной процесс.
3. Процесс выставления счета и выдачи чека.
4. Текущий процесс заключения договоров:
   - бумажный формат;
   - подписание;
   - хранение.
5. Пожелания по цифровизации:
   - ЭДО;
   - электронная подпись;
   - интеграция с 1С без вмешательства в работу бухгалтера.

### Навигация и размещение автомобилей

1. Как клиент находит свое парковочное место:
   - нумерация;
   - указатели;
   - зонирование.
2. Нестандартные ситуации:
   - место занято другим автомобилем;
   - место временно недоступно из-за ремонта.
3. Процедура разрешения конфликтов:
   - обращение к охраннику;
   - предложение временного парковочного места.

### Выезд с парковки

1. Процесс выезда:
   - по проксимити-карте;
   - вручную через охранника.
