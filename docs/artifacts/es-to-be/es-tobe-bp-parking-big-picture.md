# ES TO-BE — Big Picture процесса «Предоставление парковочного места»

## 1. Назначение файла

Этот файл фиксирует **уровень Big Picture** для процесса «Предоставление парковочного места» в терминах Event Storming:

- **Событие**
- **Вопросы, риски**
- **Актор**
- **Внешняя система**
- **Термин** (ключевые доменные понятия)

Подробные подпроцессы и модель предметной области описаны отдельно в:

- `es-tobe-bp-parking-main.md`
- `es-tobe-bp-parking-subprocesses.md`

Здесь мы только «раскладываем» основную сквозную историю по событиям и контексту, без погружения в команды, агрегаты и бизнес‑правила.
