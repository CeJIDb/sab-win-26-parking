## Состояние

Тексты из .docx извлечены в .txt; на их основе заполнены файлы `transcript-1-1.md` ... `transcript-6.md`. В каждом .md: шапка (версия, дата, ссылка на протокол), ключевые темы/цитаты и решения для быстрого поиска. Полные дословные стенограммы остаются в соответствующих `.txt` в этой папке (для 11.02 — объём большой, в .md вынесена терминология и основные решения).

## Использование как контекста

- При формулировании FR/NFR и Use Cases можно ссылаться на конкретные реплики и уточнения из транскрибаций.
- Протоколы содержат сжатые решения; транскрибации — полный ход обсуждения и нюансы.
