### Кандидаты ФТ (иерархия Роль.Объект.Действие)

### Реестр идентификаторов требований (FR-*)

Обоснование: для трассируемости и верифицируемости каждое требование из списка кандидатов получает уникальный идентификатор формата `FR-*`.

- **FR-TARIFF-001** `Тариф.ПросмотретьСписок`
- **FR-TARIFF-002** `Тариф.Просмотреть`
- **FR-TARIFF-003** `Тариф.Создать`
- **FR-TARIFF-004** `Тариф.Изменить`
- **FR-TARIFF-005** `Тариф.Деактивировать`
- **FR-TARIFF-006** `Тариф.Создать.СД.идентификатор`

## КлиентФЛ

- Тариф.ПросмотретьСписок

  Система должна предоставлять пользователю с ролью КлиентФЛ возможность просматривать список тарифов, при этом:

  - в списке отображаются атрибуты:
    - идентификатор
    - наименование
    - типТарифа
    - льготнаяКатегория
    - ставка
    - шагТарификации
    - максимальнаяСумма
    - грейсПериод
    - действуетС
    - действуетПо
    - индивидуальный
    - статус
  - доступны фильтры по:
    - типТарифа
    - льготнаяКатегория
    - статусу
    - действуетС
    - действуетПо

- Тариф.Просмотреть

  Система должна предоставлять пользователю с ролью КлиентФЛ возможность просматривать детальную карточку Тариф, содержащую:

  - все атрибуты сущности Тариф из словаря данных
  - при наличии — перечень связанных льготных категорий и/или льготных документов

## Системы

- Тариф.Создать.СД.идентификатор

  Система должна при создании новой записи Тариф:

  - автоматически формировать значение атрибута `идентификатор`
  - обеспечивать, чтобы `идентификатор` был целым числом > 0
  - гарантировать уникальность значения `идентификатор`
  - не допускать последующего изменения `идентификатор`

### Проверка полноты набора ФТ

- Основные потоки UC-13.1-13.6 покрыты:
  - создание тарифа
  - просмотр тарифа
  - изменение тарифа
  - деактивация тарифа
  - просмотр списка тарифов
- Потенциально отсутствующие ФТ (рекомендации к расширению):
  - ФТ по публикации и архивированию тарифа в терминах статусов из реестра UC
  - ФТ по управлению применимостью тарифа к типам зон
  - ФТ по версионированию тарифа и истории изменений
