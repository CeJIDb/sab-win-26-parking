# Импортированный словарь данных из BuildIn

Рабочая версия словаря данных, импортированная из BuildIn и приведенная к формату репозитория.
Документ сохраняет исходное содержимое с минимальной редактурой оформления и не является каноничным источником истины для целевой ER-модели.

## Назначение

Документ фиксирует исходный экспорт словаря данных, который команда подготовила в BuildIn.
Он полезен как промежуточный контекст для сверки идей по сущностям, атрибутам и ограничениям перед переносом решений в каноничные ERD-артефакты.

## Контекст источника

- Статус: импортированный вспомогательный материал
- Источник: экспорт из BuildIn
- Уровень зрелости: черновые и частично несогласованные предложения
- Правило использования: при расхождениях приоритет у документов в этой папке

Замена персональных имен на GitHub-аккаунты выполнена по контексту репозитория и рабочим артефактам команды.
